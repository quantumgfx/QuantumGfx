# Quantum Graphics

Quantum Graphics is a low level, cross platform, graphics API, written in C++ 17.

## Supported Platforms

- Windows (32 bit)
- Windows (64 bit)
- Linux

## Graphics backends currently in developement

- Vulkan 1.2 (Referred to in code as "Vulkan")

## Planned backends

- Vulkan 1.0 (Referred to as "VulkanLegacy")
- MoltenVk
- DirectX 12
- Metal
